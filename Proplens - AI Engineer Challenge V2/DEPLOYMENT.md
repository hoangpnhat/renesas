# Deployment Guide - Render

## Quick Deploy to Render

### Option 1: One-Click Deploy (Recommended)

1. **Tạo tài khoản Render** (nếu chưa có): https://render.com/

2. **Click vào nút Deploy to Render:**
   - Truy cập: https://dashboard.render.com/select-repo?type=web
   - Chọn repository: `hoangpnhat/Proplens---AI-Engineer-Challenge-V2`

3. **Cấu hình Web Service:**
   ```
   Name: property-sales-assistant
   Environment: Python 3
   Branch: main
   Build Command: pip install -r requirements.txt && python manage.py migrate && python scripts/load_data.py
   Start Command: gunicorn config.wsgi:application
   ```

4. **Thêm Environment Variables:**

   Trong phần "Environment Variables", thêm:

   | Key | Value |
   |-----|-------|
   | `OPENAI_API_KEY` | `sk-proj-...` (API key của bạn) |
   | `SECRET_KEY` | `django-prod-secret-key-12345` (random string) |
   | `DEBUG` | `False` |
   | `ALLOWED_HOSTS` | `property-sales-assistant.onrender.com` |
   | `PYTHON_VERSION` | `3.10.19` |

5. **Deploy:**
   - Click "Create Web Service"
   - Đợi 5-10 phút để build

6. **Truy cập ứng dụng:**
   - URL: `https://property-sales-assistant.onrender.com`
   - API Docs: `https://property-sales-assistant.onrender.com/api/docs`
   - Admin: `https://property-sales-assistant.onrender.com/admin`

### Option 2: Manual Deploy via Dashboard

#### Bước 1: Kết nối GitHub

1. Đăng nhập Render: https://dashboard.render.com/
2. Click "New +" → "Web Service"
3. Chọn "Build and deploy from a Git repository"
4. Click "Connect account" để kết nối GitHub
5. Chọn repository: `hoangpnhat/Proplens---AI-Engineer-Challenge-V2`

#### Bước 2: Cấu hình Service

Điền các thông tin sau:

**Basic Settings:**
- **Name:** `property-sales-assistant`
- **Region:** Singapore (hoặc gần bạn nhất)
- **Branch:** `main`
- **Runtime:** `Python 3`

**Build & Deploy:**
- **Build Command:**
  ```bash
  pip install -r requirements.txt && python manage.py migrate && python scripts/load_data.py
  ```
- **Start Command:**
  ```bash
  gunicorn config.wsgi:application
  ```

**Instance Type:**
- Chọn **Free** (hoặc Starter nếu cần performance tốt hơn)

#### Bước 3: Environment Variables

Click "Advanced" → "Add Environment Variable", thêm:

```bash
# Required
OPENAI_API_KEY=sk-proj-YOUR_KEY_HERE
SECRET_KEY=your-random-secret-key-here-change-this
DEBUG=False

# Auto-generated by Render
ALLOWED_HOSTS=property-sales-assistant.onrender.com

# Optional
PYTHON_VERSION=3.10.19
```

**Lưu ý:** Thay `YOUR_KEY_HERE` bằng OpenAI API key thực của bạn.

#### Bước 4: Deploy

1. Click "Create Web Service"
2. Render sẽ bắt đầu build (5-10 phút)
3. Monitor logs để kiểm tra progress

#### Bước 5: Verify Deployment

Sau khi deploy xong, test các endpoint:

```bash
# Health check
curl https://property-sales-assistant.onrender.com/api/health/

# Create conversation
curl -X POST https://property-sales-assistant.onrender.com/api/conversations/

# API Documentation
# Mở browser: https://property-sales-assistant.onrender.com/api/docs
```

## Troubleshooting

### Build Fails

**Lỗi: "No module named 'X'"**
- Kiểm tra `requirements.txt` có đầy đủ dependencies
- Solution: `pip freeze > requirements.txt` và commit lại

**Lỗi: "Memory exceeded"**
- Free tier có giới hạn RAM
- Solution: Upgrade lên Starter plan ($7/month)

### Runtime Issues

**Lỗi: "Application failed to start"**
- Kiểm tra logs: Dashboard → Service → Logs
- Common issues:
  - OPENAI_API_KEY chưa set
  - ALLOWED_HOSTS sai
  - Database migration failed

**Lỗi: "500 Internal Server Error"**
- Check logs để xem error cụ thể
- Thường do:
  - Missing environment variables
  - Database connection issues
  - ChromaDB path issues

### Database Issues

**Lỗi: "no such table: core_project"**
- Migration chưa chạy
- Solution: Xem logs, re-deploy với build command đầy đủ

**Data không load được**
- CSV file quá lớn
- Solution: Pre-commit database file hoặc load data từ external source

## Post-Deployment Setup

### 1. Create Admin User

```bash
# SSH vào Render container (nếu có access)
python manage.py createsuperuser --username admin --email admin@example.com
```

Hoặc thêm vào build command:
```bash
echo "from django.contrib.auth import get_user_model; User = get_user_model(); User.objects.create_superuser('admin', 'admin@example.com', 'admin123')" | python manage.py shell
```

### 2. Configure Custom Domain (Optional)

1. Dashboard → Service → Settings → Custom Domain
2. Add your domain: `api.yourdomain.com`
3. Update DNS records theo hướng dẫn của Render
4. Update `ALLOWED_HOSTS` với domain mới

### 3. Setup HTTPS (Auto)

Render tự động cấp SSL certificate (Let's Encrypt)
- HTTPS enabled by default
- Auto-renewal

### 4. Monitor Performance

- Dashboard → Service → Metrics
- Xem: CPU usage, Memory, Response time
- Setup alerts nếu cần

## Scaling & Optimization

### Database Optimization

**Nâng cấp từ SQLite sang PostgreSQL:**

1. Add PostgreSQL database trong Render
2. Update `settings.py`:
   ```python
   import dj_database_url
   DATABASES = {
       'default': dj_database_url.config(
           default='sqlite:///db.sqlite3',
           conn_max_age=600
       )
   }
   ```
3. Add `dj-database-url` vào requirements.txt
4. Re-deploy

### Caching

**Add Redis for caching:**

1. Add Redis service trong Render
2. Install: `pip install django-redis`
3. Configure in settings.py
4. Cache API responses

### Performance

**Tips:**
- Use Starter plan ($7/month) thay vì Free
- Enable auto-scaling
- Use CDN cho static files
- Implement rate limiting

## Cost Estimation

### Free Tier
- **Cost:** $0/month
- **RAM:** 512 MB
- **Limitations:**
  - Spins down after 15 minutes inactivity
  - Slower cold starts
  - Limited compute

### Starter Plan
- **Cost:** $7/month
- **RAM:** 512 MB
- **Benefits:**
  - Always on
  - Better performance
  - No cold starts

### Standard Plan
- **Cost:** $25/month
- **RAM:** 2 GB
- **Best for:** Production use
